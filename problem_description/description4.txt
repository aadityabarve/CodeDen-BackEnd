Given an array A[] consisting 0s, 1s and 2s. The task is to write a function that sorts the given array.
The functions should put all 0s first, then all 1s and all 2s in last.

<b>Input Format:</b>
The first line contains and integer T, which indicates the number of test cases.
The first line of each of the test contains an integer N indicating the number of elements in the array.
The next line contains N space separated integers.

<b>Output Format</b>
For each test case print the sorted array without spaces in between.

<b>Sample Input 1:</b>
1
6
0 1 2 0 1 2

<b>Sample Output 1:</b>
001122

<b>Sample Input 2:</b>
1
7
2 2 2 2 1 1 0

<b>Sample Output 2:</b>
0112222
